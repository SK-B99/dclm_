# DCLM Church Website

The official website for Deeper Christian Life Ministry (DCLM), designed to keep members and visitors informed about sermons, events, announcements, and church activities.

**Live site:** [DCLM Church](https://dclm-alpha.vercel.app)

---

## Pages

| Page | Description |
|------|-------------|
| Home | Welcome page with service times and quick links |
| About | Church history, vision, and mission |
| Sermons | Published sermons with audio and video |
| Events | Upcoming programs and activities |
| Announcements | Latest news and updates |
| Gallery | Photos and videos from services |
| Contact | Church location and contact details |

---

## Built with

- **Next.js 16** — Web framework
- **Tailwind CSS** — Styling
- **TypeScript** — Language
- **Vercel** — Hosting and deployment

---

## Installation

```bash
git clone https://github.com/SK-B99/dclm-frontend.git
cd dclm-frontend
npm install
npm run dev
```

Visit `http://localhost:3000` in your browser.
